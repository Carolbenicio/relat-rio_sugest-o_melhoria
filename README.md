## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa  Abstergo Industries , realizado por Caroline Benício. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2 (Elastic Compute Cloud )
-  **Foco da ferramenta:** Reduzir os gastos de estruturas físicas por instancias virtuais escalonáveis. Utilizando tais instancias para reduzir os custos e máquinas ociosas usando apenas o que a empresa precisa em horários de pico. 
- **Caso de uso**: Migração de aplicações legadas para as instancias EC2 com AutoScalling para utilizar apenas o que é necessário em horários de maior movimentação na aplicação dessa forma maximizando a economia 

Etapa 2: 
- Amazon S3 (Simple Storage Service)
- **Foco da ferramenta:** Otimização de custos de armazenamento de dados através de políticas de ciclo de vida
- **Caso de uso:**  Configuração do S3 para mover arquivos que não são acessados a mais de 30 dias para classes de armazenamento mais baratas reduzindo drasticamente a fatura de storage sem perder dados 

Etapa 3: 
- AWS Lambda 
- **Foco da ferramenta:** Migração para a arquitetura Servelles para eliminar a necessidade de maquinário ligado 24/7 
- **Caso de uso:** Execuçõa de scripts de automação, processamento de imagens e rotinas de backend que antes eram realizados em servidores fixos



## Conclusão
A implementação de ferramentas na empresa *Abstergo Industries*  tem como esperado redução de custos e melhorar o funcionamento da empresa, o que aumentar a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
